Osprey -> WebAssembly

Two Osprey flavors. One browser-native analytics app.

Osprey compiles a real program to wasm32-wasip1. The module emits a SQLite schema, seed data, analytics queries, and pure Osprey gold-standard metrics. The page then runs the SQL in sql.js and reconciles both engines in your browser.
