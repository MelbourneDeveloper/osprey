**Description:** Function declaration keyword. Used to define functions with parameters and return types.

## Example

```osprey
fn add(a, b) = a + b

fn greet(name) = print("Hello, " + name)
```

```osprey-ml
add (a, b) = a + b

greet name = print ("Hello, " + name)
```
