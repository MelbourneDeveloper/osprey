**Description:** Variable declaration keyword. Used to bind values to identifiers. Variables are immutable by default in Osprey.

## Example

```osprey
let name = "Alice"
let age = 25
let isActive = true
```

```osprey-ml
name = "Alice"
age = 25
isActive = true
```
