**Signature:** `abs(value: int) -> int`

**Description:** Returns the absolute value of an integer.

## Parameters

- **value** (int): The integer whose magnitude to take

**Returns:** int

## Example

```osprey
let d = abs(0 - 5)  // 5
```

```osprey-ml
d = abs 0 - 5  // 5
```
