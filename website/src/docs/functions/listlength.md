**Signature:** `listLength(list: List<t0>) -> int`

**Description:** Returns the number of elements in a list. O(1).

## Parameters

- **list** (List<t0>): The list

**Returns:** int

## Example

```osprey
listLength([1, 2, 3])  // 3
```
