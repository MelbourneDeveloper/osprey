**Signature:** `toLowerCase(s: string) -> string`

**Description:** ASCII-aware lowercase.

## Parameters

- **s** (string): The string to transform

**Returns:** string

## Example

```osprey
toLowerCase("HELLO")  // "hello"
```

```osprey-ml
toLowerCase "HELLO"  // "hello"
```
