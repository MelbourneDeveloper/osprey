---
title: "toString (Function)"
description: "Converts a value to its string representation."
---

**Signature:** `toString(value: any) -> string`

**Description:** Converts a value to its string representation.

## Parameters

- **value** (any): The value to convert to string

**Returns:** string

## Example

```osprey
let str = toString(42)
print(str)  // Prints: 42
```
