**Signature:** `mapKeys(map: Map<t0, t1>) -> List<t0>`

**Description:** All keys of the map as a list. Order unspecified.

## Parameters

- **map** (Map<t0, t1>): The map

**Returns:** List<t0>

## Example

```osprey
mapKeys(m)  // List<K>
```

```osprey-ml
mapKeys m  // List<K>
```
