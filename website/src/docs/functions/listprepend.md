**Signature:** `listPrepend(list: List<T>, value: T) -> List<T>`

**Description:** Returns a new list with value at the front. O(n).

## Parameters

- **list** (List): The list
- **value** (any): Value to prepend

**Returns:** List

## Example

```osprey
listPrepend([2, 3], 1)  // [1, 2, 3]
```
