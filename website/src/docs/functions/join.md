**Signature:** `join(parts: List<string>, separator: string) -> string`

**Description:** Concatenates parts with separator between each pair.

## Parameters

- **parts** (List<string>): Strings to join
- **separator** (string): Separator string

**Returns:** string

## Example

```osprey
join(["a","b","c"], "-")  // "a-b-c"
```

```osprey-ml
join (["a","b","c"], "-")  // "a-b-c"
```
