**Signature:** `trimEnd(s: string) -> string`

**Description:** Removes trailing whitespace.

## Parameters

- **s** (string): The string to trim

**Returns:** string

## Example

```osprey
trimEnd("  hi  ")  // "  hi"
```

```osprey-ml
trimEnd "  hi  "  // "  hi"
```
