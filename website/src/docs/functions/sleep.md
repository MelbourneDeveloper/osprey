**Signature:** `sleep(milliseconds: int) -> Unit`

**Description:** Pauses execution for the specified number of milliseconds.

## Parameters

- **milliseconds** (int): Number of milliseconds to sleep

**Returns:** Unit

## Example

```osprey
sleep(1000)  // Sleep for 1 second
print("Awake!")
```

```osprey-ml
sleep 1000  // Sleep for 1 second
print "Awake!"
```
