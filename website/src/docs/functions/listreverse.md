**Signature:** `listReverse(list: List<t0>) -> List<t0>`

**Description:** Returns a new list in reverse order.

## Parameters

- **list** (List<t0>): The list

**Returns:** List<t0>

## Example

```osprey
listReverse([1, 2, 3])  // [3, 2, 1]
```
