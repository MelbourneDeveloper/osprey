**Signature:** `listReverse(list: List<T>) -> List<T>`

**Description:** Returns a new list in reverse order.

## Parameters

- **list** (List): The list

**Returns:** List

## Example

```osprey
listReverse([1, 2, 3])  // [3, 2, 1]
```
