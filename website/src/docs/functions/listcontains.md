**Signature:** `listContains(list: List<t0>, value: t0) -> bool`

**Description:** True iff some element equals value. O(n).

## Parameters

- **list** (List<t0>): The list
- **value** (t0): Value to find

**Returns:** bool

## Example

```osprey
listContains([1, 2, 3], 2)  // true
```

```osprey-ml
listContains ([1, 2, 3], 2)  // true
```
