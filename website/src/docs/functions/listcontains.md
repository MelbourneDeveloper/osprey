**Signature:** `listContains(list: List<T>, value: T) -> bool`

**Description:** True iff some element equals value. O(n).

## Parameters

- **list** (List): The list
- **value** (any): Value to find

**Returns:** bool

## Example

```osprey
listContains([1, 2, 3], 2)  // true
```
