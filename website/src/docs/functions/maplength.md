**Signature:** `mapLength(map: Map<K, V>) -> int`

**Description:** Returns the number of entries in a map. O(1).

## Parameters

- **map** (Map): The map

**Returns:** int

## Example

```osprey
mapLength({"a": 1, "b": 2})  // 2
```
