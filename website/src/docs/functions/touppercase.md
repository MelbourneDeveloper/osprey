**Signature:** `toUpperCase(s: string) -> string`

**Description:** ASCII-aware uppercase. Unicode simple case mapping is a future addition.

## Parameters

- **s** (string): The string to transform

**Returns:** string

## Example

```osprey
toUpperCase("hello")  // "HELLO"
```
