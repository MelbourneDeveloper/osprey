**Signature:** `mapValues(map: Map<t0, t1>) -> List<t1>`

**Description:** All values of the map as a list. Order matches mapKeys.

## Parameters

- **map** (Map<t0, t1>): The map

**Returns:** List<t1>

## Example

```osprey
mapValues(m)  // List<V>
```

```osprey-ml
mapValues m  // List<V>
```
