**Signature:** `mapValues(map: Map<K, V>) -> List<V>`

**Description:** All values of the map as a list. Order matches mapKeys.

## Parameters

- **map** (Map): The map

**Returns:** List

## Example

```osprey
mapValues(m)  // List<V>
```
