**Signature:** `listConcat(left: List<t0>, right: List<t0>) -> List<t0>`

**Description:** Returns left ++ right. Same as left + right.

## Parameters

- **left** (List<t0>): Left operand
- **right** (List<t0>): Right operand

**Returns:** List<t0>

## Example

```osprey
listConcat([1, 2], [3, 4])  // [1, 2, 3, 4]
```

```osprey-ml
listConcat ([1, 2], [3, 4])  // [1, 2, 3, 4]
```
