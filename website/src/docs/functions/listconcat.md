**Signature:** `listConcat(left: List<T>, right: List<T>) -> List<T>`

**Description:** Returns left ++ right. Same as left + right.

## Parameters

- **left** (List): Left operand
- **right** (List): Right operand

**Returns:** List

## Example

```osprey
listConcat([1, 2], [3, 4])  // [1, 2, 3, 4]
```
