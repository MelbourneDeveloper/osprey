**Description:** A sequence of characters representing text. Supports string interpolation and escape sequences.

## Example

```osprey
let greeting = "Hello, World!"
let name = "Alice"
let message = "Hello, ${name}!"
```

```osprey-ml
greeting = "Hello, World!"
name = "Alice"
message = "Hello, ${name}!"
```
